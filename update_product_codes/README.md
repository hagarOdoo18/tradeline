# update_product_codes
Module to update product.product e_code and gs1_code by barcode using an Excel file upload wizard.

## Installation
1. Copy this module to your Odoo addons path.
2. Restart Odoo server.
3. Update apps list and install 'Update Product Codes by Barcode'.

## Usage
- Go to Products -> Update Product Codes
- Upload Excel file with headers: Barcode, e_code, gs1_code
- Click Update.
